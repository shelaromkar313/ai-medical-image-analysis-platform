# AI-Powered Medical Image Analysis & Clinical Decision Support System

## Overview

A production-ready enterprise healthcare platform that combines Java Full Stack Development, Artificial Intelligence, Machine Learning, Deep Learning, Computer Vision, and Cloud Deployment.

## Technology Stack

### Frontend
- React.js 18
- TypeScript
- Material UI (MUI)
- Tailwind CSS
- Redux Toolkit
- React Router
- Axios
- Recharts

### Backend
- Java 21
- Spring Boot 3
- Spring Security with JWT
- Spring Data JPA
- MySQL
- Redis
- Maven

### AI/ML Service
- Python 3.11
- FastAPI
- TensorFlow
- PyTorch
- OpenCV
- Scikit-learn
- Grad-CAM
- SHAP

### Cloud & DevOps
- AWS (EC2, S3, RDS)
- Docker
- Docker Compose
- Nginx
- GitHub Actions

## Core Features

- 🔐 JWT-based Authentication & RBAC
- 👨‍⚕️ Doctor Dashboard
- 👤 Patient Dashboard
- 🖼️ Medical Image Upload (X-ray, MRI, CT, etc.)
- 🧠 AI-powered Image Classification
- 🌡️ Grad-CAM Heatmap Visualization
- 📄 Automated Medical Report Generation
- 📊 Analytics & Trend Dashboards
- 📥 PDF Report Downloads
- 🔔 Notifications System
- 📝 Audit Logs
- 🔍 Search & Filters
- 🌙 Dark Mode Support

## Supported Medical Image Types

- Chest X-rays
- Brain MRI
- CT Scans
- Skin Lesion Images
- Retinal Images

## Deep Learning Models

- ResNet50
- DenseNet121
- EfficientNet
- MobileNetV3
- Vision Transformer (ViT)
